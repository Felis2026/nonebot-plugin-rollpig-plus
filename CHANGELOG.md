# 更新日志

## v0.7.3

### 独立发布准备

- 将 Felis-dev整理为独立仓库与独立包名 `nonebot-plugin-rollpig-plus`。
- Python 模块名调整为 `nonebot_plugin_rollpig_plus`，避免与原作包名直接冲突。
- 更新插件元数据、README、CHANGELOG、预览图与配置示例，补齐面向 GitHub / PyPI / NoneBot 商店的发布说明。

### 内置资源更新

- 将公共资源包内置进插件默认资源，默认小猪数量更新至 149 只。
- 同步最新 `pig_rules.json`，内置熟食类、售罄类等资源规则。
- 保持私有资源 overlay 默认关闭，仍可通过配置手动启用。

## v0.7.2

### 稳定性修复

- 加固本地 `pig_data.json` 读取与写入流程，避免坏 JSON 被静默覆盖。
- 加固云端资源同步，改为流式下载、单文件大小上限与临时目录激活。
- 加强资源包校验，下载失败时保留旧缓存或回退到内置资源。
- 修复今日小猪检查与保存之间的并发竞态，降低重复抽取风险。

### 性能与 AI 边界

- 新增 HTML/Chromium 渲染总并发预算，避免多类生图任务互相打爆。
- 图片版小猪图鉴引入常驻页面池与缓存上限。
- AI 烤猪新增并发、token、输出字符数与超时边界。
- 默认 AI 并发调整为 4，兼顾可用性与请求堆积风险。

## v0.7.1

### PigHub 兼容性修复

- 修复 PigHub 旧接口失效导致的 `找猪` / `搜猪` 不可用问题。
- 新增新旧 PigHub 接口兼容。
- 优化搜索失败时的异常兜底。

## v0.7.0

### 图片版小猪图鉴

- 新增 `小猪图鉴` 图片命令。
- 使用 HTML/CSS 渲染图鉴内容层，并与底图合成。
- 新增图鉴聚合快照、EX Lv. 展示、NEW / MAX 标记。

### 配置与性能

- 新增 `rollpig_config.json` / `config/rollpig.json` 非敏感配置文件支持。
- 保留 `.env` / NoneBot 配置对敏感项和部署覆盖项的优先级。
- 新增图鉴渲染缓存、渲染超时与缩放配置。

### 烤群友充能

- 新增普通烤群友充能存储。
- 支持多 Bot 状态同步时共享充能状态。

## v0.6.2

### 私有资源 overlay

- 支持公有资源包之上的私有资源 overlay。
- 支持私有资源 manifest 与 token 配置。

### 特殊形态

- 新增更多特殊形态兼容。
- 熟食类、吃掉了、人类、售罄等形态参与烤猪拦截。

## v0.6.1

### 云端资源同步

- 新增云端小猪资源包同步。
- 支持从 `manifest.json` 拉取 `pig.json`、`pig_rules.json` 与图片资源。
- 新增超管命令 `同步小猪资源` / `刷新小猪图鉴`。

## v0.6.0

### 新功能

- 新增 EX Lv. 成长机制。
- 新增猪圈进度摘要。
- 新增烤群友玩法与更多特殊文案。

### 云端同步

- 新增 rollpig-cloud 存储后端。
- 支持今日小猪、图鉴成长状态、烤群友充能等状态跨 Bot 同步。

## 更早版本

更早版本主要继承自原作插件与早期 Felis 分支，包含基础今日小猪、随机小猪、找猪、本周小猪、烤猪与资源扩展等功能。
